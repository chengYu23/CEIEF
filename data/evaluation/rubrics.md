# CEIEF Evaluation Rubrics
## 文化教育交互增强框架 - 评分量表

---

## 1. 评分维度概览

| 维度 | 英文名称 | 权重 | 评分范围 |
|---|---|---|---|
| 历史语境匹配度 | Historical Context Matching | 33% | 1-5 |
| 文化标签对齐度 | Cultural Label Alignment | 33% | 1-5 |
| 语言风格模态对齐 | Language Style Modal Alignment | 34% | 1-5 |

---

## 2. 维度一：历史语境匹配度（Historical Context Matching）

**定义**：响应内容与所涉及文化事件、人物、实践的历史语境的准确匹配程度。

| 分值 | 描述 |
|---|---|
| 5 | 响应包含至少两个具体历史细节，历史背景完全准确，无时代错误 |
| 4 | 响应包含一个具体历史细节，历史背景基本准确，有轻微不精确 |
| 3 | 历史背景模糊，使用泛化陈述，缺乏具体时代依据 |
| 2 | 历史信息不完整或有明显错误，但总体方向正确 |
| 1 | 历史信息严重错误或完全缺失历史背景 |

**评分指引**：
- 检查是否提及具体朝代/时期（如战国时期、唐代、周代）
- 检查人物描述是否与历史记载一致
- 检查文化实践的起源描述是否准确

---

## 3. 维度二：文化标签对齐度（Cultural Label Alignment）

**定义**：响应中使用的文化概念标签与知识图谱中已验证的文化关系的一致程度。

| 分值 | 描述 |
|---|---|
| 5 | 响应中的文化标签完全对应KG三元组，关系描述准确，概念边界清晰 |
| 4 | 大部分文化标签对应KG，有1-2处轻微偏差但不影响核心理解 |
| 3 | 部分文化标签对应KG，存在概念混淆或关系描述不精确 |
| 2 | 少数文化标签对应KG，主要依赖通用文化描述 |
| 1 | 文化标签与KG几乎无对应，使用非专业或错误的文化概念 |

**评分指引**：
- 检查核心文化概念是否正确使用（如区分"孝道"与"尊老"的层次关系）
- 检查文化关系方向是否正确（如端午节embodies集体记忆，而非相反）
- 检查价值导向标签的使用是否符合文化语境

---

## 4. 维度三：语言风格模态对齐（Language Style Modal Alignment）

**定义**：响应的语言风格与所设定角色、任务类型和学习者水平的适配程度。

| 分值 | 描述 |
|---|---|
| 5 | 语言风格完全符合角色设定，教学风格与任务类型高度匹配，难度适当 |
| 4 | 语言风格基本符合角色，有轻微风格偏差但不影响交互质量 |
| 3 | 语言风格部分符合，存在角色外语言或不适当的正式/非正式切换 |
| 2 | 语言风格较少符合角色设定，多处出现风格不一致 |
| 1 | 语言风格完全不符合角色设定，缺乏文化教育适切性 |

**角色风格指引**：
- **museum_guide（博物馆导览员）**：生动叙事，具体案例，亲切友好
- **cultural_teacher（文化教师）**：结构清晰，引导性强，重视理解深度
- **historical_interpreter（历史解说员）**：专业严谨，时代感强，注重史实
- **value_mediator（价值调解员）**：中立平衡，多视角呈现，避免价值判断偏颇

---

## 5. 综合评分计算

```
综合得分 = (历史语境匹配度 × 0.33) + (文化标签对齐度 × 0.33) + (语言风格模态对齐 × 0.34)
```

| 综合得分范围 | 质量等级 | 描述 |
|---|---|---|
| 4.5 - 5.0 | 优秀 | 高质量文化教育交互，可作为示范案例 |
| 3.5 - 4.4 | 良好 | 基本满足文化教育要求，有小幅改进空间 |
| 2.5 - 3.4 | 一般 | 文化教育效果有限，需要系统改进 |
| 1.5 - 2.4 | 较差 | 文化教育效果不足，需要重新设计 |
| 1.0 - 1.4 | 不合格 | 无法满足文化教育基本要求 |

---

## 6. 评分员信度协议

### 6.1 评分流程
1. 每位评分员独立完成评分，不进行事前讨论
2. 每个响应由两位评分员独立打分
3. 计算评分员间信度（Cohen's Kappa）
4. 分歧超过1分时，启动第三评分员仲裁

### 6.2 信度标准
- **目标 Kappa**：κ ≥ 0.70（实质性一致）
- **可接受 Kappa**：0.60 ≤ κ < 0.70
- **需要重新标定**：κ < 0.60

### 6.3 评分员培训
- 提供10个校准案例（每维度覆盖5个分值水平）
- 进行评分讨论，统一评分标准理解
- 正式评分前进行预评分测试，达标后方可参与正式评分

---

## 7. 预标注提示词使用说明

预标注仅供参考，不作为最终评分依据。
人工评分员应独立判断，不受预标注结果影响。
如预标注与人工评分差异超过2分，需在记录表中注明原因。
